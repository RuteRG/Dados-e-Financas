# 📊 Dados e Finanças

## 🎯 Contexto e Objetivos
Este caderno temático tem como objetivo explorar a relação entre **análise de dados** e **finanças**, utilizando Python e técnicas de Machine Learning para compreender padrões e apoiar decisões financeiras.  
Os objetivos principais são:
- Aprender fundamentos de análise de dados com Python.
- Revisar conceitos de finanças pessoais e investimentos.
- Aplicar modelos de Machine Learning em previsões financeiras.

## 📒 Miniguia de Estudo

### 🔹 Introdução à análise de dados com Python
- Uso de bibliotecas como `pandas`, `numpy` e `matplotlib`.
- Limpeza e manipulação de dados financeiros.

### 🔹 Educação Financeira Básica: A REGRA DOS 3 FATORES
- 50% para gastos essenciais (moradia, alimentação, transporte).
- 30% para investimentos/poupança (reserva de emergência, aposentadoria).
- 20% para lazer/consumo (viagens, hobbies).

### 🔹 Machine Learning aplicado a previsões financeiras
- Modelos de regressão para prever preços de ativos.
- Séries temporais e previsão de tendências.

## 📖 Glossário
- **ROI (Return on Investment)**: Retorno sobre investimento.  
- **Diversificação**: Estratégia de reduzir riscos distribuindo investimentos.  
- **Série temporal**: Conjunto de dados ordenados cronologicamente.
